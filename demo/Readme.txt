This package includes the following two directories:

 - columbus // Stata do-file and the Columbus crime data (Anselin, 1988).
 - jpmuni_crime // Stata do-file and sample data of crime and unemployment
                   at the municipality level in Japan.

See Kondo (2016) for further information.


References:
Kondo, Keisuke (2016) "Introduction to spatial econometric analysis: 
   Creating spatially lagged variable in Stata," Mimeo.
